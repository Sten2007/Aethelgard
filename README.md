# ⚔️ Aethelgard — Shadows of the Abyss

A browser-based pixel-art RPG built with vanilla JavaScript and Vite.

🎮 **[Play the game here](https://Sten2007.github.io/Aethelgard/)**

---

## 🗺️ About

Aethelgard is a top-down RPG where you explore six distinct worlds, fight powerful guardians, collect legendary gear, and ultimately face the **Void Watcher** — the final boss lurking at the edge of eternity.

---

## 🕹️ How to Play

| Key | Action |
|-----|--------|
| `W A S D` | Move your character |
| **Yellow tile** | Enter Town — shop for spells & armor |
| **Purple tile** | Teleporter — travel to the next world |
| **Black tile** | Boss Lair — fight the world guardian |
| `Space / Click` | Advance text / select in menus |
| `Escape` | Close the Guide |

---

## ⚔️ Classes

| Class | Strength | Special Passive |
|-------|----------|-----------------|
| **Knight** | Balanced fighter | None |
| **Berserker** | High ATK | +20% damage |
| **Rogue** | High SPD | Critical hit chance |
| **Mage** | Spell caster | -50% MP cost |
| **Paladin** | High DEF tank | +5 HP regen per turn |

## 🌍 The Six Worlds

| World | Name | Boss |
|-------|------|------|
| I | Whispering Forest | Stone Guardian |
| II | Inferno Peaks | Inferno Phoenix |
| III | Crystal Wastes | Crystal Hydra |
| IV | Void Abyss | Nether Fiend |
| V | Celestial Spire | Celestial Dragon |
| VI | Realm of the Watcher | **Void Watcher** *(Final Boss)* |

---

## 🧙 Spells & Loot

- **30 spells** across all 6 worlds — damage, healing, and **lifesteal** types
- Enemies have a rare chance to drop **armor**
- Bosses always drop **legendary equipment**
- Check the in-game **📜 Game Guide** for full drop tables and spell lists

---

## 🛠️ Running Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📦 Built With

- **Vanilla JavaScript** (ES Modules)
- **HTML5 Canvas** for rendering
- **Vite** for bundling
- **Web Audio API** for procedural chiptune music
